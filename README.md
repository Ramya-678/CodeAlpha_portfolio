# Personal Portfolio Website

## Project Description
This is a *Personal Portfolio Website* developed using HTML, CSS, and JavaScript.  
The website showcases my profile, skills, projects, and contact details in a clean and professional layout.  
It is created as part of the *“Build Your Own Portfolio Site”* task.

## Features
- About Me section  
- Skills section  
- Projects section  
- Contact details (GitHub, LinkedIn, Email)  
- Responsive and clean UI design  
- Easy navigation  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Project Structure
portfolio  
│  
├── index.html  
├── style.css  
├── script.js (optional)  

## How to Run the Project
1. Download or clone the repository  
2. Open the project folder  
3. Open the file named index.html  
4. The portfolio website will open in your web browser  

## Output
The webpage displays a personal portfolio website.  
Users can view profile details, skills, projects, and contact information.

- Improved HTML and CSS layout skills  
- Gained experience in linking GitHub and LinkedIn profiles  
- Practiced responsive web design

This project helped me build my own portfolio website and improve my front-end development skills.
